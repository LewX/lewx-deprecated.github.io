1 安装Hugo 打开一个终端 Terminal，输入与操作系统对应的 Hugo 安装命令 brew install hugo 安装完成后，输入以下命令来确认 hugo version 2 新建网站 进入你想存放 Hugo 网站文件夹的目录，如 cd ~/Documents 执行以下命令新建一个 Hugo 网站 hugo new site myblog #myblog是网站文件夹名 选择主题 打开 https://themes.gohugo.io 页面，选择一个你喜欢的主题，如blackburn 将选择的主题clone至本地 #进入网站目录 cd ~/Documents/myblog #创建"themes"目录 mkdir -p themes #进入"themes"目录 cd themes #将blackburn主题克隆至"blackburn"目录 git clone https://github.com/yoshiharuyamashita/blackburn.git blackburn 编辑配置文件 cd ~/Documents/myblog echo theme = \"blackburn\" >> config.
